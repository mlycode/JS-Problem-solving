Countdown Recursive Code

Recursive Code

Definition: Code which executes itself. Continuosly pushes functions onto the call stack to be executed until the Base Case is reached. Each time the recursive code is called, a different input is used.

Code

run countDown(num) in the console

  • Uses recursive code and not a loop (countDownLoop(num) uses a loop)
  • countDown(num) calls itself continuously until it reaches a defined base condition (if (num <=0) return)
