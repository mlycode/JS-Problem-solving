Sum range Recursive Code

Recursive Code

Definition: Code which executes itself. Continuosly pushes functions onto the call stack to be executed until the Base Case is reached. Each time the recursive code is called, a different input is used.

Code

run sumRange(num) in the console

  • sumRange(3) returns: return 3 + sumRange(2) (first return is now waiting for sumRange(2) to execute on the call stack)
  • sumRange(2) returns: return 2 + sumRange(1) (now sumRange(2) is waiting for sumRange(1))
  • Once sumRange(1) returns 1, sumRange(2) returns 2 + 1, and finally sumRange(3) returns 3 + 2 + 1). Returning 1 is the base case
