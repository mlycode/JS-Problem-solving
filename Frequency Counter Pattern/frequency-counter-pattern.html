Frequency Counter Pattern - Open the Console

  • Uses objects or sets to collect values/frequencies of values
  • can often avoid the need for nested loops with arrays/strings O(n^2) operations

Example

Write a function called same, which accepts two arrays. Function should return true if every value in the first array has it's corresponding value squared in the second array. Frequency of values must be the same.

same([1,2,3]), [4,1,9] // true

same([1,1,3]), [1,9,9] // false (must be same frequency)

To run code in console:

  • sameNaive(array1, array2)
    • Solution has a nested loop, results in a time complexity of O(n^2) which is not ideal
  • sameRefactor(array1, array2)
    • Solution has non-nested loops, results in time complexit of O(n), better than nested loops

Exercise

Given two strings, write a function to determine if the second string is an anagram of the first. Solve without using a nested loop (O(n) time complexity).

To run code in console:

  • validAnagram(string1, string2)
