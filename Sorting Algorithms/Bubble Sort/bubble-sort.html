Bubble Sorting Algorithm

The largest values bubble to the top, one at a time.

Two values are compared, and the larger value is moved forward/swapped. The pass repeats until the data is sorted. Each loop moves the largest item to the end of the data set, so the data set decreases by one each time.

Code

run bubbleSort([numbers]) in the console

  • Solution uses a nested loop, O(n^2) time complexity
